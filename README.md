# Terraria
# How to use:
This cannot be hosted as a static website; it needs to be hosted as a web server.
1. Fork this repository
2. Go to render.com, connect the GitHub account that you forked this with
3. In your dashboard, click the new button, then the web server button.
4. Put in this repo for your selected web server.
5. For the build command, put "npm install", and for the start command, put "node server.js"
6. Make sure the Instance you put is the free version; chances are, you don't want to be spending money on making this work.
7. Click Deploy web service, and you should be good. Just wait for it to finish making the site.
